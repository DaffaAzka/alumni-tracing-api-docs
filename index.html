Authentication

To access the API, you must include an Authorization header in the format below:

Authorization: Bearer {your_token}
Content-Type: application/json
Accept: application/json

Endpoints

1. Get All Alumni

URL: /api/alumni

Method: GET

Status: 200

Authorization: Required

Description: Retrieves all alumni data. Supports pagination and optional name-based search.

Request Parameters:

  • current-page (int): The current page number.
  • name (string): Search by alumni name (optional).

Response Example:

{
  "message": "Mengambil semua data alumni untuk page 2",
  "total_pages": 2,
  "data": [
    {
      "number": 11,
      "id": 14,
      "full_name": "Wadi Situmorang S.Pt",
      "major": "Petani / Pekebun",
      "status": "Bekerja"
    },
    {
      "number": 12,
      "id": 16,
      "full_name": "Lulut Saragih",
      "major": "Tukang Batu",
      "status": "Bekerja"
    }
  ]
}

3. Create Alumni

URL: /api/alumni

Method: POST

Status: 201

Authorization: Not Required

Description: Creates new alumni data. Guests are allowed only 2 submissions per day. If exceeded, IP will be blocked for 24 hours.

Fields:

  • full_name (string): Alumni's full name.
  • place_of_birth (string): Place of birth.
  • date_of_birth (date): Date of birth (YYYY-MM-DD format).
  • major_id (int): Major ID.
  • graduation_year (year): Graduation year (YYYY format).
  • status (string): Current status (e.g., Bekerja, Kuliah).
  • status_detail (string): Detailed status.

Response Example:

{
  "message": "data berhasil di buat",
  "data": {
    "id": 3,
    "full_name": "Candrakanta Wasita",
    "place_of_birth": "Pariaman",
    "date_of_birth": "1986-10-02",
    "major": "Tukang Jahit",
    "graduation_year": "2014",
    "status": "Bekerja",
    "status_detail": "Ut perferendis culpa atque delectus est."
  }
}

An Excel file named Alumni.xlsx will be downloaded.
